# express

## Use it

```shell
# clone it

git clone https://github.com/skels/express.git my-web-site

# create `my-web-site` repository on GitHub
# do not initialize the new repository with README, license, or gitignore files

git remote set-url origin https://github.com/k33g/my-web-site.git

# make changes and then eg:

git add README.md 
git commit -a
git push -u origin master
```
